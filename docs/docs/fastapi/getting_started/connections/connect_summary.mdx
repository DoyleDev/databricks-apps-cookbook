# Connecting to Databricks Apps

There are several options to consider when connecting to a deployed Databricks App.

### Connectivity Options
| Option | Use-Case |
| -------- | ------- |
| [Local Machine to Databricks App](/docs/fastapi/getting_started/connections/connect_from_local) | Development & testing, exploratory work   |
| [External App to Databricks App](/docs/fastapi/getting_started/connections/connect_from_external) | External integration with Databricks services |
| [Databricks App to Databricks App](/docs/fastapi/getting_started/connections/connect_from_app) | Internal consumption of Databricks services across workspaces (e.g. app federation) |
