# Connecting to Databricks Apps

There are several options to consider when connecting to a deployed Databricks App.

### Connectivity Options
| Option | Use-Case |
| -------- | ------- |
| Local Machine to Databricks App  | Development & testing, exploratory work   |
| External App to Databricks App  | External integration with Databricks services |
| Databricks App to Databricks App  | Internal consumption of Databricks services across workspaces (e.g. app federation) |
