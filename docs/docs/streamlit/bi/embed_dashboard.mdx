# Embed a dashboard

This recipe embeds a [Databricks AI/BI dashboard](https://docs.databricks.com/aws/en/dashboards/) into a Databricks App. 

## Code snippet

```python title="app.py"
import streamlit.components.v1 as components
        
iframe_source = "https://workspace.azuredatabricks.net/embed/dashboardsv3/dashboard-id"

components.iframe(
    src=iframe_source,
    height=600,
    scrolling=True
)
```

Copy and paste the dashoard embedding URL from the dashboard UI **Share** -> **Embed iframe**.
